DOWNLOAD ANACONDA
-----------------
[cloudera@quickstart ~]$ cd Downloads/

AFTER INSTALLATION:
------------------
[cloudera@quickstart ~]$ vi .bashrc

---------------------------------

STEPS TO INSTALL JAVA on LINUX
_______________________________________________________________________________________________________________________


DOWNLOAD JDK (.rpm)
-------------------
[cloudera@quickstart Downloads]$ sudo rpm -ivh jdk-8u201-linux-x64.rpm


AFTER INSTALLATION:
------------------
[cloudera@quickstart ~]$ vi .bashrc


ENTER THE BELOW LINE, IN THE FILE
---------------------------------
JAVA_HOME="/usr/java/jdk1.8.0_201-amd64/"
export PATH=$JAVA_HOME/bin:$PATH


TO VERIFY
---------
[cloudera@quickstart ~]$ echo $PATH
